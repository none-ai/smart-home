# Smart Home

A smart home control system for managing IoT devices.

## 🎯 Features

- Device management (lights, thermostats, locks, cameras)
- Scheduling and automation
- Energy monitoring
- Mobile app support
- Voice control integration
- Real-time status updates
- Security alerts

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/none-ai/smart-home.git
cd smart-home

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```

Visit http://localhost:5000

## 📋 Requirements

- Python 3.8+
- Flask
- MQTT broker
- IoT device integrations
- See `requirements.txt` for full list

## 🛠️ Tech Stack

- Python 3
- Flask
- MQTT
- JavaScript
- IoT Protocols
